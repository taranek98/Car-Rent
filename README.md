Instrukcja instalacji:
- Pobierz projekt z GitHub. Możesz użyć komendy w terminalu: git clone https://github.com/taranek98/Car-Rent.git
- Zainstaluj potrzebne paczki NuGet używajac komendy w terminalu: dotnet restore
- Do pliku konfiguracyjnego dodaj ConnectionString aby połączyć aplikacje z bazą danych
- Do pliku konfiguracyjnego dodaj JWT
